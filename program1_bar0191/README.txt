HOW TO RUN:

1. compile server and client

	-> javac Client.java
	-> javac Server.java

2. Run server with port number as argument

	-> java Server portno
	-> example: java Server 5000

3. Run client with desired hostname and port number as arguments

	-> java Client hostname portno
	-> example: java Client cse01.cse.unt.edu 5000

4. Follow client prompt
